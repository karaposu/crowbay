# Core Technical Concepts

## 1. Frame Extraction Pipeline
Intelligent sampling of video frames based on visual change detection and task requirements.
   - **1a. Event Expectation Scripts**: Flexible action sequences that guide adaptive sampling density
   - **1b. OpenCV Scene Change Detection**: Computer vision pre-processing to identify transitions and segment video

## 2. Visual Context Aggregation
Combining insights from multiple frames to understand complete action sequences.

## 3. LLM Vision Integration
Efficient batching and prompting strategies for vision-language models.

## 4. Task Specification Language
Structured format for defining what constitutes successful task completion.

## 5. Evidence Extraction System
Capturing and storing proof artifacts from analyzed video content.

## 6. Confidence Scoring Framework
Multi-factor assessment of verification reliability and uncertainty quantification.

## 7. Temporal Event Detection
Identifying and ordering significant moments within video timeline.

## 8. API Gateway Architecture
Request handling, queuing, and response management for video processing.
